        public string PrintOXs(int n)

            var curr = 0;
            var output = new StringBuilder();

            {
                var temp = Math.Pow(2, i);

^는 xor연산임. (같으면 0, 다르면 1)

k<<1=k*2  k<<2=k*4  k<<3=k*8
